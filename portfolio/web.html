Web Maps

Where Are All The Cabs? (Hexbins)

This web map visualizes the distribution of available taxis in Singapore in real time. Live API data is retrieved from Data.gov.sg and binned in hexagons using d3.js and Leaflet.

Where Are All The Cabs? (Heatmap)

This web map visualizes the distribution of available taxis in Singapore in real time. Live API data is retrieved from Data.gov.sg and displayed as a heatmap using Leaflet heatmap plugin.

WalkViz:Redmond

This project intends to assess the walkability potential within the city limit of Redmond, Oregon. The interface allows you to interactively visualize how walkable a particular spot in Redmond is. This interactive map is completed as part of Geography 591: Advanced GIS coursework at the University of Oregon.

Jet's Journey

This web map animates the longest known, and probably the most studied, migration of mule deer in the United States. “Jet” is just one of a few thousands deer that annually migrate from their summer to winter range. Migration data is courtesy of InfoGraphics Lab and Wyoming Migration Initiative.

Greater Yellowstone Ecosystem Mule Deer Migration

This interactive map shows the movement of mule deer along routes used during the spring migration from winter ranges up into the mountains. Part of Wyoming Migration Initiative project.

140 Years of UO

This 140 Years of UO web map allows users to interactively visualize the history of current campus buildings of the University of Oregon and learn how these buildings come together to where they are today. This web map lets users to virtually navigate around campus to access the rich history of University of Oregon architecture.

UW-Madison Food Map

This project intends to educate our community about the variety of sustainable dining options that are available on UW-Madison campus. The research was conducted by undergraduates at Nelson Institute and the findings are implemented in a user-friendly web map.

Tambopata: Who Owns Paradise?

This project provides an interactive map to support undergraduate students in Geography 339 to complete their Tambopata assignment. Other cartographers involved are Grace White and Mikaela Weisse. A paper about this project was published in Journal of Maps in spring 2014. Read it here.

Demography of Canada

This project visualizes the changes of population in major Canadian cities from 1951 to 2011.

England Livestock Map

This project illustrates the difference of variety farm production in England by counties.

Penang Tourism Map

This map shows the places of interest in the island of Penang in Malaysia.
