web map


This 140 Years of UO web map allows users to interactively visualize the history of current campus buildings of the University of Oregon and learn how these buildings come together to where they are today. The UO Libraries website provides such rich information that it requires multiple platforms to dynamically envision the data effectively. Using open-source geospatial and non-geospatial technologies, such as Leaflet, jQuery and other JavaScript libraries, this web map is made available to any internet user. Even if you are not familiar with the University of Oregon area, this web map allows you to virtually navigate around campus to access the rich history of U of O's architecture. To use the map, click on the web map link in the navigation bar and explore! If you have any questions or suggestions, feel free to send me an e-mail at rudyo@uoregon.edu.

This web map is developed by Rudy Omri , a graduate student in the Department of Geography, University of Oregon.
